># `ADK Flask` *(single user, multi sessions)*
>
>```bash
>python -m venv .venv
>
>source .venv/bin/activate      # macOS/Linux
>.\.venv\Scripts\activate       # PowerShell
>.\.venv\Scripts\activate.bat   # Windows CMD
>
>pip install -r .\requirements.txt
>python app.py
>```
